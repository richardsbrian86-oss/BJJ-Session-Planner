🥋 BJJ Session Planner
A full‑stack scheduling application designed for Brazilian Jiu‑Jitsu instructors and academies.
This tool automates instructor availability, prevents calendar conflicts through external calendar integration, and streamlines booking confirmations — reducing admin work so instructors can focus on teaching.

🚀 Features
Instructor Availability Management  
Instructors set their available times; the system handles scheduling logic.

Calendar Conflict Prevention  
Integrates with external calendars to ensure no double‑booking.

Automated Booking Confirmations  
Students receive instant confirmation when a session is approved.

Simple, Fast UI  
Built for quick navigation and minimal friction during scheduling.

Scalable Full‑Stack Architecture  
Designed to grow with more instructors, classes, and students.

🧠 Why This Project Exists
As a Brazilian Jiu‑Jitsu black belt and instructor, I’ve seen how chaotic scheduling can get:

Instructors forgetting to update availability

Students double‑booking sessions

Endless back‑and‑forth messages

No centralized view of who is teaching and when

This app solves those real‑world problems with automation and clarity.
